Introduction

This example shows how to incorporate different 3D models and movie texture for tracking. Using GPS trakcing configuration, this tutorial negates the necessity of a tracking pattern. The user can take a picture and set it as the tracking pattern for the second stage -- dummy tracking. In this setting, the user can put 3D models onto the picture taken. 3 gestures have been included and enabled in this tutorial, namely the drag, pinch and rotation gesture. A screenshot can also be saved by pressing of a button.

Learn

Use GPS tracking and set the coordinate offset. Load geometries and assign them to different groups for gesture handling. Comprehensive gesture control of 3D geometries. Save screenshot and reactivate the camera for tracking.
